On behalf of the python-dev team, release manager [Georg Brandl](http://pythonic.pocoo.org/) has announced the final release of [CPython 3.2.1](http://www.python.org/download/releases/3.2.1/). Windows installers and tarballs are available as of July 10, so please consider upgrading to this release.

The [What's New](http://docs.python.org/3.2/whatsnew/3.2.html) document lists all of the new features in 3.2, and the [Misc/NEWS](http://hg.python.org/cpython/file/v3.2.1/Misc/NEWS) file in the source lists each bug fixed.

If you find any issues with this release or any other, please report them to [http://bugs.python.org/](http://bugs.python.org/).
