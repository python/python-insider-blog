The first release candidate for Python 3.4, [Python 3.4.0rc1,](http://www.python.org/download/releases/3.4.0/) has been issued on February 10 2014. This is a preview release of the next major release of Python, Python 3.4, and is not yet suitable for production environments.  
The new features added to Python 3.4 are described in more detail in [What's New in Python 3.4](http://docs.python.org/3.4/whatsnew/3.4.html).
