The first release candidate of Python 2.7.11, the next release in the 2.7.x series is now available. Please see the [download page](https://www.python.org/downloads/release/python-2711rc1/) to obtain 2.7.11rc1 for testing. If no serious problems are detected, the final release will happen in two weeks.
