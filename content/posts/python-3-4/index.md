Python 3.5.0rc1 is now [available for download.](https://www.python.org/downloads/release/python-350rc1/) This is a preview release, and its use is not recommended for production settings. However, as a "release candidate", it should be very similar to the final product. Python 3.5.0 final is scheduled for release in mid-September.
