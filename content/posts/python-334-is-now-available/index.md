A new maintenance release, [Python 3.3.4](http://www.python.org/download/releases/3.3.4/), was released on February 9 2014. This release fully supports OS X 10.9 Mavericks. In particular, this release fixes an issue that could cause previous versions of Python to crash when typing in interactive mode on OS X 10.9.
