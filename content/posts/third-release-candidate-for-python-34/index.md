The [third release candidate for Python 3.4 has been released](http://www.python.org/download/releases/3.4.0/). You can also retrieve the source code using Mercurial from the [http://hg.python.org/releasing/3.4/](http://hg.python.org/releasing/3.4/). [What's New in Python 3.4](http://docs.python.org/3.4/whatsnew/3.4.html) has been updated with the latest changes and is closed to its final draft form.
